# Mostly used commands

## Creating virtualenv
```
virtualenv --python=/usr/bin/python3.8 venvnew
```

## Uninstalling CUDA


For CUDA 10.1 or newer, try:
```
sudo /usr/local/cuda/bin/cuda-uninstaller
```

For CUDA 10.0, try:
```
sudo  /usr/local/cuda/bin/uninstall_cuda_10.0.pl
```
